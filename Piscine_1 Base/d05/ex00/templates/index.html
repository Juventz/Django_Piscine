Ex00: Markdown Cheatsheet

This page provides a complete reference for Markdown syntax with examples and explanations.

1. Headers

Use # for headers. Add more # for smaller headers:

# Header 1
## Header 2
### Header 3

2. Emphasis

Use * or _ for emphasis:

*italic* or _italic_
**bold** or __bold__
***bold italic*** or ___bold italic___

3. Lists

Unordered and ordered lists:

Unordered

- Item 1
- Item 2
  - Subitem 1
  - Subitem 2

Ordered

1. Item 1
2. Item 2
   1. Subitem 1
   2. Subitem 2

4. Links

Inline and reference-style links:

[inline link](https://example.com)
[reference link][1]

[1]: https://example.com

5. Images

Add images with !:

![Alt text](https://example.com/image.jpg)

6. Code Blocks

Inline code uses backticks (`):

`inline code`

Fenced code blocks use triple backticks:

```
Code block
```

7. Blockquotes

Use > for blockquotes:

> This is a blockquote.
>> Nested blockquote.

8. Tables

Create tables with pipes (|) and dashes (-):

| Header 1 | Header 2 |
|----------|----------|
| Data 1   | Data 2   |
| Data 3   | Data 4   |

9. Horizontal Rules

Use three or more dashes or asterisks for horizontal rules:

---
***
___

10. Escaping Characters

Use backslashes to escape Markdown syntax:

\*literal asterisks\*

11. Task Lists

Use - [ ] for a task list:

- [x] Completed task
- [ ] Incomplete task

12. Strikethrough

Use ~~ for strikethrough:

~~strikethrough~~

Conclusion

This cheatsheet covers most of the common Markdown syntax elements. Feel free to explore and practice!
